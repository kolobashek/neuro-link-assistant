Информация о работе с моделями

Что такое Hugging Face?

Hugging Face - это платформа для обмена моделями машинного обучения, которая предоставляет доступ к тысячам предобученных моделей для различных задач обработки естественного языка и компьютерного зрения.

Как добавить модель?

Для добавления модели вы можете:

  1. Найти модель через поиск на Hugging Face Hub
  2. Нажать кнопку "Добавить" рядом с найденной моделью
  3. Заполнить форму с информацией о модели
  4. Нажать кнопку "Добавить" для сохранения модели в системе

Требования к моделям

Для корректной работы с системой модели должны соответствовать следующим требованиям:

  • Быть доступными на Hugging Face Hub
  • Поддерживать генерацию текста (для моделей типа "chat" и "completion")
  • Иметь совместимый с transformers формат
